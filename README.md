# Pre-Analisis-estadistico
Realiza un análisis estadistico preventivo de una un archivo de excel (.xlsx) para visualizar por medio de una Gráfica de control el comportamiento de los datos.

# Guía de uso del archivo de ensayo
Este archivo de Excel está diseñado para evaluar resultados de lotes en comparación con límites establecidos para cada variable. Se organiza en dos hojas principales:

1. Hoja “Límites”
Contiene los valores de referencia para cada variable.

Columnas:

Variable: nombre de la característica a evaluar (ejemplo: pH, valoración de activo).

Límite inferior: valor mínimo aceptable.

Límite superior: valor máximo permitido.

Estos límites sirven como criterios de aceptación para los resultados de los lotes.

2. Hoja “Datos”
Registra los resultados obtenidos en cada lote.

Columnas:

Lote: identificador único (ejemplo: 002XY, 003XY).

Variables: resultados medidos (pH, valoración de activo, etc.).

Cada fila corresponde a un lote y muestra los valores que se compararán contra los límites definidos en la hoja anterior.

🔎 ¿Qué se puede hacer con este archivo?
Evaluar conformidad: verificar si los resultados de cada lote están dentro de los límites establecidos.

Detectar desviaciones: identificar rápidamente valores fuera de rango.

Analizar tendencias: observar cómo se comportan las variables a lo largo de varios lotes.

Apoyar decisiones: usar la información para tomar acciones correctivas o validar la calidad del proceso.

✅ Instrucciones para el lector
Revise primero la hoja “Límites” para conocer los criterios de aceptación.

Consulte la hoja “Datos” para ver los resultados de cada lote.

Compare cada valor con sus límites correspondientes.

Identifique lotes conformes y lotes con desviaciones.

Use esta información como apoyo en análisis estadísticos, reportes de calidad o decisiones de proceso.
